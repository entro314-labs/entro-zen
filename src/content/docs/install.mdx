---
title: Installation
---

Install with your favorite package manager. This project uses `pnpm`.

```shell
pnpm install @entro314labs/entro-zen
```

## Usage

### 1. Import styles

```js
import '@entro314labs/entro-zen/styles.css';
```

### 2. Import components

```js
import { Button } from '@entro314labs/entro-zen';

export function Component() {
  return <Button variant="primary">Button</Button>;
}
```
