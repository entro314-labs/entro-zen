Borders are always solid and a fixed color. You can control the `radius` and `position` of borders.

## CSS variables

Border radius values are defined as a 5-step scale from smallest to largest.

<Example>
  <DataTable
    data={[
      { name: '--border-radius-1', value: '2px' },
      { name: '--border-radius-2', value: '4px' },
      { name: '--border-radius-3', value: '8px' },
      { name: '--border-radius-4', value: '16px' },
      { name: '--border-radius-full', value: '9999px' },
    ]}
  >
    <DataColumn id="name" label="Name" />
    <DataColumn id="value" label="Value" />
  </DataTable>
</Example>

Border sizes are defined as a 4-step scale from smallest to largest.

<Example>
  <DataTable
    data={[

</Example>
