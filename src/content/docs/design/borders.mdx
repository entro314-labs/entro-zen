title: Borders

Borders are always solid with a default border color. You can control the `position`, `radius`, and `color` of borders.

## Border position

  <Label>Default</Label>
  <ExampleBox border backgroundColor="2" borderRadius="0" />

  <ExampleBox border="top" backgroundColor="2" borderRadius="0" />
  <Label>Border right</Label>
  <ExampleBox border="right" backgroundColor="2" borderRadius="0" />

  <ExampleBox border="bottom" backgroundColor="2" borderRadius="0" />
  <Label>Border left</Label>

</Example>

## CSS variables

Border radius values are defined as a 5-step scale from smallest to largest.

<Example>
  <DataTable
    data={[
      { name: '--border-radius-1', value: '2px' },

      { name: '--border-radius-3', value: '8px' },
      { name: '--border-radius-4', value: '16px' },
      { name: '--border-radius-full', value: '9999px' },
    ]}
  >
    <DataColumn id="name" label="Name" />
    <DataColumn id="value" label="Value" />
  </DataTable>
</Example>
