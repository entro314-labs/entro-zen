---
title: Table
---

A table displays data in rows and columns and enables a user to navigate its contents via
directional navigation keys, and optionally supports row selection and sorting.

<Example>
  <Table>
    <TableHeader>
      <TableColumn>Name</TableColumn>
      <TableColumn>Email</TableColumn>
      <TableColumn>Date created</TableColumn>
    </TableHeader>
    <TableBody>
      <TableRow>
        <TableCell>Bob</TableCell>
        <TableCell>bob@aol.com</TableCell>
        <TableCell>5/5/2020</TableCell>
      </TableRow>
      <TableRow>
        <TableCell>Fred</TableCell>
        <TableCell>fred@yahoo.com</TableCell>
        <TableCell>4/7/2021</TableCell>
      </TableRow>
      <TableRow>
        <TableCell>Ted</TableCell>
        <TableCell>ted@gmail.com</TableCell>
        <TableCell>11/20/2010</TableCell>
      </TableRow>
      <TableRow>
        <TableCell>Joe</TableCell>
        <TableCell>joe@outlook.com</TableCell>
        <TableCell>1/18/2016</TableCell>
      </TableRow>
    </TableBody>
  </Table>
</Example>
